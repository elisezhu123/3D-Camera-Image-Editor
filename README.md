# Gemini 3D Camera Image Editor 项目文档

## 项目概述

**项目名称：** Gemini 3D Camera Image Editor  
**项目类型：** AI驱动的3D图像编辑器  
**核心功能：** 基于Google Gemini API的3D视角图像生成工具  
**技术栈：** React, Three.js, TypeScript, Vite  
**项目链接：** https://aistudio.google.com/u/2/apps/drive/12QxyxlzbTW75zBXqSdTnftEu8BwbNIK1

## 项目简介

Gemini 3D Camera Image Editor 是一款创新的AI图像编辑工具，利用Google Gemini的先进AI技术，结合3D可视化技术，允许用户通过调整相机参数来生成具有不同视角和角度的图像。该工具特别适用于需要多角度图像生成、3D视觉效果设计和创意图像处理的场景。

## 核心功能

### 1. 3D相机控制系统
- **水平偏航（Azimuth）：** 0°-360°范围内的水平旋转控制
- **垂直仰角（Elevation）：** -30°-60°范围内的垂直角度调整
- **镜头距离（Distance）：** 0.6x-1.4x范围内的缩放控制
- **视角预设：** 提供9个预设视角按钮（前后左右、四角及中心）

### 2. 图像处理功能
- **参考图像上传：** 支持最多3张参考图像的上传
- **批量生成：** 支持1-4张图像的批量生成
- **多种比例：** 支持16:9、4:3、1:1、9:16等多种画面比例
- **镜头类型：** 提供全景、中景、近景、特写四种镜头类型

### 3. 实时预览系统
- **3D可视化界面：** 实时显示相机位置和角度
- **参数实时反馈：** 实时显示当前的偏航、俯仰和焦距数据
- **生成进度追踪：** 清晰的生成进度指示器

## 技术架构

### 前端技术栈
- **React 19.2.3：** 现代化前端框架
- **Three.js 0.182.0：** 3D图形渲染引擎
- **@react-three/fiber & @react-three/drei：** React与Three.js的集成
- **TypeScript：** 类型安全的开发
- **Vite：** 快速的构建工具
- **Lucide React：** 精美的图标库

### AI服务
- **Google Gemini API：** 提供AI图像生成能力
- **Gemini API集成：** 通过@google/genai库与AI服务交互

### 构建与部署
- **Vite配置：** 支持开发服务器和生产构建
- **环境变量：** 安全管理API密钥
- **TypeScript配置：** 类型检查和编译

## 使用流程

1. **上传参考图像：** 点击上传按钮或拖拽图像到素材库
2. **调整相机参数：** 使用滑块或预设按钮调整视角
3. **设置生成选项：** 选择画面比例和生成数量
4. **开始渲染：** 点击"开始渲染"按钮生成图像
5. **下载结果：** 渲染完成后可下载生成的图像

## 部署与配置

### 本地运行
1. 安装Node.js
2. 运行 `npm install` 安装依赖
3. 设置 `GEMINI_API_KEY` 环境变量
4. 运行 `npm run dev` 启动开发服务器

### 生产部署
- 支持Vite构建系统
- 可部署到现代Web服务器
- 需要配置API密钥环境变量
